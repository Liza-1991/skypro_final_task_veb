Git

  • git status - показывает, какие файлы находятся в индексе и готовы к коммиту, какие файлы были изменены, но ещё не добавлены в индекс, какие новые файлы появились в проекте и пока не отслеживаются Git.
  • git clone - клонирование существующего репозитория.
  • git branch <имя_ветки> - создание новой ветки.
  • git commit -m "Сообщение коммита" - создание коммита. Команда переносит все изменения из индекса в репозиторий, сохраняя их как отдельный коммит.
  • git push - отправка изменений в удалённый репозиторий.
